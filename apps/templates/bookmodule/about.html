About iTask

Welcome to iTask, your comprehensive task management system.

What is iTask?

iTask is a task management system designed to help you organize, manage, and track your tasks effectively. Whether you're juggling multiple projects or just need a simple to-do list, iTask has got you covered.

Features

  • Create, Edit, and Delete Tasks: Easily manage your tasks with options to create new tasks, edit existing ones, or delete tasks that are no longer needed.
  • Add Checklists to Tasks: Break down your tasks into smaller, manageable checklist items. Add, edit, or delete checklist items as your task progresses.
  • Task Details: View detailed information for each task, including the title, creation date, deadline, priority, state, and description.

About the Project

This website is a project for a university web course, developed to demonstrate the practical application of web development skills. The project includes features such as user authentication, CRUD (Create, Read, Update, Delete) operations for tasks and checklists, and the use of modern web technologies like HTML, CSS and Django.
